# 啡快口令风格观察与仿写准备

## 当前结论

这套文案不是单一词库，而是会随时间、场景和活动更新的短句系统。当前收集到 304 条样本，其中 1 条正文转写待图像复核，已经显示出明显演变：

| 阶段 | 样本气质 | 例子 |
| --- | --- | --- |
| 2019 上线早期 | 人设、流行梗、季节画面并存 | 今日锦鲤、凌波微步、蔚蓝海面 |
| 2020-2021 | “喊出来有点羞耻”的戏剧性放大，同时加入安慰与祝福 | 重启试试、和幸福撞满杯、年年皆胜意 |
| 2022-2024 | 更接近“当天抽到的一句好运签”，也出现面向未来与自我状态的柔和动作句 | 诸事皆宜、向快乐出发、总有远方可奔赴、美好不期而遇 |
| 2025-2026 联名/官方叙事与晒图传播 | 针对场景或 IP 定制，出现英文、城市口语、歌词意象和主题视觉词池；同时出现更柔和的日常动作句 | 家是唯一城堡、爱上五月天、给生活一点留白、慢下来看世界、`take it easy.` |

## 文案构件

### 1. 语义目标

口令大致在给顾客四种东西：

| 目标 | 写法 | 样本 |
| --- | --- | --- |
| 给一个当天身份 | 名词化称号、自夸但不压人 | 今日锦鲤、全能小天才、最靓的仔 |
| 给一句轻祝愿 | 不承诺结果，只提示好运正在靠近 | 好事将至、好运加满、水逆绕行 |
| 给一个天气/季节画面 | 场景本身带安慰感 | 冬日暖阳、鸟语蝉鸣、风和景明 |
| 制造取单小戏剧 | 念出口时好玩或微微害羞 | 芝麻开门、重启试试、左边一起画个龙 |

### 2. 常见句型

| 结构 | 说明 | 真样本 |
| --- | --- | --- |
| 四字祝语 | 最稳妥，像轻量好运签 | 平安喜乐、诸事皆宜、烦恼退散 |
| `名词 + 动作/状态` | 把祝福变成正在发生的事情 | 乌云散开、好运加满、财神到家 |
| `和/给你 + 抽象好物 + 动作` | 温柔但有动作感 | 和幸福撞满杯、给你温暖阳光 |
| `季节/时间 + 小判断` | 口语化，不像广告 slogan | 喜欢夏天、新一年总晴天 |
| `自我/身份称号` | 让顾客成为主角 | 锦鲤本鲤、天选之子 |
| IP/歌词/网络梗 | 要能被立刻识别，且适合当面念 | 六脉神剑、珊瑚海、Dobby is free |

### 3. 语言约束

- 普通口令优先控制在 4 至 7 个汉字；这是 2025 年星巴克中国对媒体明确给出的产品规则。
- 语气应轻、正向、可口述；避免训诫、沉重承诺和过强销售感。
- 比起直接说“幸福”“成功”，更偏好让它发生动作：`撞满杯`、`加满`、`将至`、`退散`。
- 咖啡品牌痕迹通常不直说，偶尔通过杯饮动作或谐音自然露出，如 `和幸福撞满杯`、`美式发生`。
- 允许少量“社死感”，但应是亲切玩笑或共同文化记忆，而不是让人难堪的标签。

## 数据使用建议

- 仿写训练时先只使用 `official_*`、`reported_order`、`reported_observation` 和 `reported_image`；`reported_example` 可作参考语料而非金标准。
- 将日常、季节、节日、出行、考试/工作、联名 IP 分池生成，避免所有文案都落成泛泛好运句。
- 保留原始来源链接及截图证据。后续从社交平台新增样本时，优先记录发布日期、口令原图、场景与是否官方转发。
- `starbucks_now_passphrases.csv` 是真实样本库；未来仿写生成文本应另建文件，不与真实样本混放。

## 下一轮优先采集方向

1. 小红书话题 `星巴克啡快口令`：第一财经在 2024 年报道时称已有 5000 多篇笔记，是最大潜在样本源。
2. 星巴克中国微博与小红书账号：优先找节日、考试季、返乡和联名活动，证据等级最高。
3. 联名专项：哈利波特、五月天和周杰伦主题词池均已取得具体订单图样本；下一步可按活动名称专项查漏。
4. 城市/门店专项：官方曾明确口令会因城市、门店变化，目前公开文本样本仍不足。
