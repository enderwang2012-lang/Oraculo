# Starbucks Now 取单口令语料来源

## 统计口径

- 当前收录：304 条不重复文本样本，其中 1 条（`花开地处`）按发布者正文保留待原图核字。
- `official_text` / `official_social`：星巴克官方页面或官方账号文字直接写出的口令，可信度最高。
- `reported_order` / `reported_test` / `reported_observation`：媒体明确描述订单或现场体验得到的口令。
- `image_verified`：已经读取社媒或媒体帖子原图，订单界面中的口令清晰可辨。
- `social_text`：公开社媒正文直接说明为“啡快口令”，但尚未将配图作为逐字证据。
- `social_text_uncertain`：社媒正文明确声称是口令，但写法疑似可能存在输入误差，必须保留原样并等待图片复核。
- `reported_image` / `reported_ugc`：媒体转引晒单、展示截图或根据网友分享列举；适合研究风格，但仍应在取得原图后升级证据等级。
- `reported_example` / `article_example`：媒体文章作为真实啡快口令语境中的列举或对话示例；先纳入待核验池。

## 机制证据

- 星巴克在上线稿中称，口令由系统随机生成，用来代替简单数字编码。
- [新浪科技转述的星巴克中国分享](https://finance.sina.com.cn/tech/2021-01-07/doc-iiznctkf0636910.shtml)称：截至上线一年半，已经创造数千个口令，并会按季节、时刻、城市和门店变化。
- [青年报 2025 年报道](https://www.why.com.cn/wx/article/2025/09/25/17587770361094319548.html)引述星巴克中国称：普通啡快口令长度不超过 7 字，通常融合时下热点、季节特色与品牌内核。`Dobby is free` 是报道中的联名晒单样本，可能属于 IP 活动的特殊规则，暂不据此否定长度规则。
